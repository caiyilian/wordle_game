# Wordle Game (Web Version) | 猜单词游戏 (网页版)

这是一个基于 Python (Flask) 和 Web 技术构建的多人互动式 Wordle（猜单词）游戏。

玩家可以通过网页聊天框发送指令来开始游戏、猜测单词，并实时获取颜色反馈。

## 🎮 游戏特色

*   **互动式玩法**：通过聊天指令控制游戏，支持多人观看/参与（基于同一房间状态）。
*   **灵活设定**：支持自定义单词长度（默认为 5，可指定 3-10）。
*   **丰富词库**：内置多种英语考试词库（CET4, CET6, TOEFL, GRE 等）。
*   **智能反馈**：
    *   🟩 **绿色**：位置正确
    *   🟨 **黄色**：字母存在但位置错误
    *   ⬜ **灰色**：字母不存在
*   **提示系统**：遇到困难时可申请提示。

## 🛠️ 安装与启动

### 1. 克隆仓库

```bash
git clone https://github.com/caiyilian/wordle_game.git
cd wordle_game
```

### 2. 安装依赖

确保你已经安装了 Python 3.8+。

```bash
pip install -r requirements.txt
```

### 3. 启动游戏

```bash
python app.py
```

启动后，在浏览器访问：[http://127.0.0.1:5000/](http://127.0.0.1:5000/)

## 📖 游戏指南

所有操作均在网页右下角的聊天输入框中进行。

### 核心指令

| 指令                | 说明                          | 示例       |
| :------------------ | :---------------------------- | :--------- |
| **`猜单词`**        | 开始一局新游戏（默认 5 字母） | `猜单词`   |
| **`猜单词 <数字>`** | 开始指定长度的游戏            | `猜单词 6` |
| **`结束游戏`**      | 强制结束当前游戏并查看答案    | `结束游戏` |
| **`提示`**          | 显示当前已知的所有正确线索    | `提示`     |

### 如何猜测

*   直接在输入框发送单词（例如：`apple`）。
*   系统会自动检查单词合法性、长度以及是否重复。
*   查看上方的网格获取颜色反馈。

## 📂 项目结构

*   `app.py`: Flask 后端服务入口。
*   `game.py`: 游戏核心逻辑与词库管理。
*   `templates/`: 前端 HTML 模板。
*   `static/`: 前端 CSS 和 JavaScript 文件。
*   `resources/`: 词库与字体文件。
